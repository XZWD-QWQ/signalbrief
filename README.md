# SignalBrief

SignalBrief is a static web app for the Mind the Product hackathon. It turns messy user feedback into pain point clusters, opportunity areas, MVP priorities, user stories, and a one-page product brief.

## Try Locally

Open `index.html` in a browser.

For a pre-generated demo view, open:

```text
index.html?demo=1
```

## Devpost Copy

Elevator pitch:

```text
An AI tool that turns messy user feedback into clear product opportunities, MVP priorities, and a one-page product brief.
```

Built with:

```text
AI, Product Management, JavaScript, HTML, CSS, Novus.ai, Vercel
```

## Deployment

This is a static site. Deploy the `outputs/signalbrief` folder to Vercel, Netlify, GitHub Pages, or any static hosting service.

If the host asks for settings:

```text
Framework preset: Other / Static
Build command: leave blank
Output directory: ./
```

## Novus.ai

Before final submission, paste the Novus.ai tracking snippet in `index.html` above `app.js`, deploy the site, confirm activity appears in the Novus dashboard, then upload a dashboard screenshot to Devpost.
